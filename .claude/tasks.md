## How to Use This File

1. At the start of a Claude Code session, QuadCode reads this file automatically
2. Tell QuadCode: "Work through today's tasks" and it executes all items sequentially
3. Add tasks using the format above — be specific about org and agent
4. QuadCode will mark items [x] as it completes them and add notes below each

**Example task entry:**
```
- [ ] Build user dashboard with workout history chart | Org: TransformFit | Agent: Pixel+Vault | Repo: adapt-evolve-progress | Notes: Use Recharts, pull from sessions table
```
